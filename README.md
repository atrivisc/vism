# Vism

## Requirements

- Python 3.13.7
- PostgreSQL 17 database
- RabbitMQ 4

## Setup

1) Create and activate a virtual environment:
```bash
python3.13 -m venv .venv
source .venv/bin/activate
```
2) Install packages:
```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
